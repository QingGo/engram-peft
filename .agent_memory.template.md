## 🏗️ Architecture Decisions
- 选择在 Layer 10, 11 注入 Engram 层，平衡性能和效果
- 使用 CPU 预计算哈希索引，避免 GPU 计算开销
- 支持三种训练模式：engram_only, preserve_trainable, full_finetune

## 🤝 Human Decisions
- 优先保证与 Hugging Face 生态的兼容性
- 不修改基础模型的任何代码，仅通过 forward hooks 注入
- 所有测试必须在无GPU环境下通过

## ⚠️ Gotchas & Context
- transformers 4.40+ 版本对 DynamicCache 进行了重构
- peft 0.11+ 版本修改了 Adapter 的保存格式
- 稀疏张量在梯度累积时需要特殊处理
